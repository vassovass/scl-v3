## Outcome

The public roadmap page shows users what features are planned, in progress, and completed.

---

## What is Needed

### Columns/Phases

- **Now:** Currently being worked on
- **Next:** Coming soon
- **Later:** Planned for the future
- **Done:** Completed features

### Content Display

- Each item shows title and brief description
- Items are ordered by priority within each column
- Different types (bug, feature) are visually distinguishable

### Agent Work Indicator

- Items the AI agent is actively working on are highlighted
- Visual indicator (icon, glow, badge) makes active work obvious

### Export

- Users can export the roadmap as CSV
- Export includes: title, description, status, type

### Voting (if exists)

- Users can vote on which items they want most
- Vote counts are visible
- Voting requires being logged in

---

## Success Criteria

- [ ] Public users can see planned/in-progress/completed features
- [ ] Items the agent is working on are clearly highlighted
- [ ] Users can export roadmap data
- [ ] Page works well on mobile and desktop
